Installation

Step 1: Add office.terminal.ink to your trusted sites

This sounds really sketchy, but it's required, as we don't have enough money to pay for a code signing certificate. If you don't trust us, build the code yourself.

  1. Open the Start Menu, and open Internet Properties
  2. In the Security Tab, click on Trusted Sites, and click the Sites button
  3. Add https://office.terminal.ink to the list of sites, then Close and then click OK

Step 2: Download and install DiscordForOffice

DiscordForOffice may automatically search for updates.

Download DiscordForOffice Licence

Uninstallation

  1. Open Control Panel
  2. Click Uninstall a program
  3. Find and uninstall products

You will need to uninstall:

Name Publisher
Discord For PowerPoint Terminal.ink
Discord For Word Terminal.ink
Discord For Excel Terminal.ink
